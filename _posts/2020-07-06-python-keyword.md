Python 내 keyword와 builtin의 종류

---


### Python Keyword / Builtin
#### [Keyword] : 두꺼운 표시된 word들

~~~python
import keyword
keyword.kwlist

import
False
None
True
...
~~~

- 할당이 불가능한 word들
- shift+tab으로 정보확인 불가능
  - help('키워드이름') 문을 통해 정보확인



#### [Builtin] : 옅게 색 표시된 word들

- ex. globals, print
- 본래 내장된 함수
- import 없이 사용 가능
- 변수로 할당 가능하나, not recommended
