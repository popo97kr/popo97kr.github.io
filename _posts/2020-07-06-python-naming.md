### Naming Rules in Python
- PEP : python coding guide - ex.naming rules

- Naming rules

  1. 강제 규칙
     - 이름 앞에 숫자 X
     - '_'를 제외한 특수문자 사용 X
  2. 권장 규칙
     - **SNAKE 방식** = function을 나타냄
       - 소문자로 구성되며, '_'로 연결
       - ex. load_iris
     - **CAMEL 방식** = class를 나타냄
       - className / ClassName
       - ex. KNeighborsClassifier
